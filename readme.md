### Custom-data-structures

**Time Aware Linked list**

| Operation      | Time Complexity                             |
| -------------- | ------------------------------------------- |
| Insert at tail | O(1)                                        |
| Search by time | O(n) if arbitrary, but can often stop early |

**Secure HashTable**

| Operation | Secure Hash Table |
| --------- | ----------------- |
| Insert    | O(1) average      |
| Lookup    | O(1) average      |
